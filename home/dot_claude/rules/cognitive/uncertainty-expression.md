# Uncertainty Expression

Express uncertainty explicitly based on confidence level:

| Confidence | Expression |
|------------|------------|
| 0.8+ | Assertive OK ("This is...") |
| 0.5-0.8 | "Probably...", "Likely..." |
| 0.3-0.5 | "Might be...", "Needs verification" |
| <0.3 | "Uncertain...", "Hypothesis..." |

**Prohibited:** "definitely"/"absolutely" without evidence

**Format:**
```
[Judgment] **Confidence: X.X** - Rationale: [why]
```
